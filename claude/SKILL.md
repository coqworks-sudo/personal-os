# SKILL — Layer 2エージェントの処理能力

## 担当できること
- Slackチャンネルの新着メッセージ収集・分類
- themes/各ファイルへの気づき追記
- synthesis/synthesis.mdの接続仮説更新
- todo/log/YYYY-MM-DD.md の活動ログ生成
- weekly.mdの進捗更新
- GitHubへのコミット・push（[routine]プレフィックス）
- #dailyへの日次通知

## 担当できないこと
- seeds/の新規エントリ作成（対話必須）
- weekly.mdのタスク追加・削除（対話必須）
- Slack App/Botの設定変更
